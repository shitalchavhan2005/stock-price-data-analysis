# 📈 Stock Price Data Analysis

## 📌 Project Overview

This project demonstrates the complete Data Analysis workflow using Python.

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization

---

## 📂 Dataset

- Stock Prices Dataset
- Contains historical stock market data for multiple companies.

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Project Workflow

### 1. Data Cleaning
- Loaded dataset
- Checked missing values
- Removed null values
- Removed duplicates
- Converted date column

### 2. Exploratory Data Analysis
- Dataset overview
- Summary statistics
- Company analysis
- Trading volume analysis
- Correlation analysis

### 3. Data Visualization
- Bar Chart
- Line Chart
- Histogram
- Pie Chart
- Heatmap

---

## 📈 Results

The dataset was successfully cleaned and analyzed.

Important business insights were extracted using EDA and visualized using different charts.
