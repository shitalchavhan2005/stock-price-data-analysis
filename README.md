# 📖 Project Description

This project focuses on analyzing historical stock market data using Python. The objective was to clean the raw dataset, perform exploratory data analysis (EDA), and create meaningful visualizations to understand stock price trends and trading patterns.

---

# 🎯 Objectives

- Load the stock market dataset
- Handle missing values
- Remove duplicate records
- Convert data into appropriate formats
- Perform Exploratory Data Analysis (EDA)
- Create meaningful visualizations
- Export the cleaned dataset

---

# 📂 Dataset Information

The dataset contains historical stock prices of multiple companies.

### Dataset Columns

| Column | Description |
|---------|-------------|
| Symbol | Company Symbol |
| Date | Trading Date |
| Open | Opening Price |
| High | Highest Price |
| Low | Lowest Price |
| Close | Closing Price |
| Volume | Trading Volume |
